# 🚀 ORBITAL X

## Sistema Inteligente de Monitoramento Espacial Experimental

ORBITAL X é uma aplicação web interativa desenvolvida para simular o monitoramento operacional de uma missão espacial experimental fictícia.

O sistema permite acompanhar dados ambientais e operacionais em tempo real através de um dashboard dinâmico e interativo.

---

# 📌 Objetivo

O objetivo do projeto é representar uma central inteligente de monitoramento espacial capaz de acompanhar operações científicas em ambientes extremos do sistema solar.

A plataforma realiza simulações de:
- temperatura;
- radiacao;
- comunicação orbital;
- energia;
- combustível;
- sensores ambientais.

---

# 🌌 Ambientes Monitorados

- Titã
- Marte
- Europa
- Lua

---

# 🔬 Operações Científicas

O sistema possui diferentes tipos de operações:

- Monitoramento Atmosférico
- Análise de Radiação
- Pesquisa Climática
- Comunicação Orbital

Cada operação altera dinamicamente os dados exibidos no dashboard.
